Exemplos de text-shadow

A propriedade text-shadow: 2px 2px 4px; define uma sombra ao redor do texto, e é composta por quatro valores que controlam a aparência da sombra. Aqui está a explicação de cada parte:

2px (horizontal offset)

Este valor define o deslocamento da sombra no eixo horizontal (esquerda ou direita).

  • Se for positivo, a sombra será deslocada para a direita do texto.
  • Se for negativo, a sombra será deslocada para a esquerda do texto.

2px (vertical offset)

Este valor define o deslocamento da sombra no eixo vertical (para cima ou para baixo).

  • Se for positivo, a sombra será deslocada para baixo do texto.
  • Se for negativo, a sombra será deslocada para cima do texto.

4px (blur radius - raio de desfoque)

Este valor define o quanto a sombra será "borrada" ou "desfocada".

  • Se for 0, a sombra não terá desfoque, ou seja, será uma linha nítida.
  • Quanto maior o valor, mais desfocada ou suave será a sombra. No exemplo, com 4px, a sombra terá um desfoque suave.

Cor (opcional)

A cor da sombra pode ser definida após o valor de desfoque, utilizando qualquer valor de cor válido em CSS, como rgba(0, 0, 0, 0.5) para uma sombra preta semitransparente ou red para uma sombra vermelha. Se não for especificada, o navegador usará a cor padrão, geralmente preto.
