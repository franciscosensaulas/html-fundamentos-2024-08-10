Exemplo de Diferentes Text Decoration

Text-decoration: none

Este texto não tem decoração. Usamos text-decoration: none para remover qualquer decorações, como o sublinhado padrão de links.

Text-decoration: underline

Este texto está sublinhado.

Text-decoration: overline

Este texto tem uma linha acima.

Text-decoration: through

Este texto tem uma linha através dele.

Text-decoration: underline overline

Este texto tem uma linha embaixo e outra em cima.
